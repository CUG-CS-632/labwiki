# 实验室 Wiki

这里整理 632 实验室常用设备、服务和工作流程的使用说明。文档的目标是让新成员能快速完成服务器选择、远程连接、科研计算和文件传输等基础操作，也让已有成员能查到服务器配置、访问地址和排障方法。

## 从这里开始

| 需求 | 入口 |
|------|------|
| 选择使用哪台服务器 | [服务器](/server) |
| 查看 Windows GPU 服务器配置 | [Windows GPU 服务器](/server/windows-server) |
| 查看 Linux CPU 服务器配置 | [Linux CPU 服务器](/server/linux-server) |
| 图形化连接服务器桌面 | [远程桌面](/server/remote-desktop) |
| 命令行或 VSCode 远程连接 | [SSH 与 VSCode](/server/ssh-vscode) |
| 访问共享文件夹 | [SMB 文件共享](/server/smb) |
| 配置 Python 环境和镜像源 | [Python 环境](/server/python) |

## 文档结构

目前文档按资源类型组织：

- **[服务器](/server)**：两台实验室服务器的定位、硬件配置、连接方式和文件共享说明

后续新增内容建议继续按“资源概览 → 使用指南 → 规范与排障”的顺序组织，避免把配置、教程和临时说明混在同一页。
